Configuración Git Ignore

  1. Crear el archivo .gitignore en el directorio raíz de tu proyecto.
  2. Abrir el archivo y añadir los nombres de los archivos o directorios que deseas ignorar.
  3. Guardar los cambios en el archivo .gitignore.
  4. Agregar y confirmar el archivo .gitignore al repositorio con los comandos git add .gitignore y git commit -m "Añadir archivo .gitignore".
  5. Verificar la configuración utilizando git status para asegurarte de que los archivos ignorados no aparecen.
  6. Actualizar el .gitignore según sea necesario a medida que el proyecto crece.
  7. Con estos pasos tu gitignore debe funcionar a la perfeccion, en caso tal de que no funcione, aqui tienes un enlace donde esta el paso a paso mas detallamente.
